## 프로젝트 개요

React, Vue 등의 프레임워크 없이 브라우저 네이티브 API만으로 Web Components기반 SPA(Single Page Application)를 구현한 프로젝트입니다.

## 주요 기능

- **SPA 라우팅**: History API 기반의 클라이언트 사이드 라우팅
- **게시글 목록**: 카테고리별 필터링 및 무한 스크롤 (Intersection Observer API)
- **게시글 상세**: 게시글 본문 및 댓글 조회
- **사이드바 네비게이션**: 카테고리(전체 / 개발 / 공지) 전환

## 라우트 구조

| 경로        | 설명                      |
| ----------- | ------------------------- |
| `/`         | 전체 게시글 목록          |
| `/dev`      | 개발 카테고리 게시글 목록 |
| `/notice`   | 공지 카테고리 게시글 목록 |
| `/post/:id` | 게시글 상세 페이지        |

## 실행 방법

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm start
```
